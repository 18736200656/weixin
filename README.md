nodejs + socket.io 的多人在线聊天后台系统，前台项目地址：[前台项目地址](https://github.com/bailichen/vue-weixin)

### 项目运行：

```
git clone https://github.com/bailicangdu/node-weixin

cd node-weixin

npm install

npm run dev (需先开启mongodb)

访问: http://localhost:8003

```
